1. A Survey of Behavioural Authentication in Internet of Things Ecosystems

Context: Carleton University (April 10, 2023)[cite: 1]

Abstract Overview: This survey evaluates the landscape of securing access to personal and shared IoT devices by leveraging user behaviour (what a person does) via built-in sensors (e.g., accelerometers, microphones)[cite: 1]. It explores machine learning classifiers—primarily Support Vector Machines (SVM) and Decision Trees/Random Forests—for seamless user authentication through gait, gesture, and voice analytics[cite: 1].

Key Challenges Highlighted: High computational demands, lack of standardized performance guidelines, vulnerability to mimicking or membership inference attacks, and data privacy vulnerabilities[cite: 1].

Machine Learning Behaviour-based Biometrics Continuous Authentication Gait/Gesture/Voice

2. About Nguyen et al.'s vCCA security

Context: Technical Analysis / Literature Critique (December 2023)[cite: 2]

Abstract Overview: A rigorous evaluation of the breakthrough paper "Fully Homomorphic Encryption beyond IND-CCA1 Security: Integrity through Verifiability" by Mark Manulis and Jerome Nguyen[cite: 2]. The report details VCCA (Verifiable Chosen Ciphertext Attack) security, a stronger semantic security framework applicable to malleable Fully Homomorphic Encryption (FHE) architectures[cite: 2].

Key Components Evaluated: The use of Succinct Non-Interactive Arguments of Knowledge (SNARKs) to prove the validity of homomorphic evaluations, alongside a critique regarding the real-world performance overheads of these protocols[cite: 2].

Fully Homomorphic Encryption (FHE) VCCA Security Semantic Security SNARKs

3. DOLO: Data Over Light Onboarding for Smart Smoke Detectors

Context: University of Ottawa Research Project[cite: 3]

Abstract Overview: Proposes DOLO (Data Over Light Onboarding), a novel physical-layer onboarding protocol designed to replace unreliable, vulnerable SoftAP (temporary Wi-Fi hotspot) setup models for smart home sensors[cite: 3].

Protocol Mechanism: Utilizing a physical smartphone flash to beam encrypted Wi-Fi configuration credentials via m-level or On-Off light encoding directly into an ambient light sensor (using the Google Nest Protect as a reference architecture)[cite: 3]. DOLO significantly slashes the device attack surface while overcoming common iOS/Android network switching constraints[cite: 3].

Smart Home IoT Data-over-Light (DoL) Photovoltaic Sensors Alternative Onboarding
